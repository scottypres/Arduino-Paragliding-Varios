# Arduino Paragliding Varios

This repository contains two mini vario hardware variants:

- `Adafruit Feather Mini Vario/`
- `Sparkfun ESP32 WROOM Vario/`

Each variant keeps its KiCAD projects, FlatCAM outputs, Arduino sketches, and supporting notes in its own folder. The KiCAD `Component References` folders are duplicated per variant so project-local `${KIPRJMOD}/../Component References/...` 3D model references continue to resolve when a project is opened from its own folder.

The GitHub remote may still need to be renamed from `Vario_Adafruit_Featherwing` to `Arduino Paragliding Varios` or a URL-safe equivalent such as `Arduino-Paragliding-Varios`.
